# Outputs

This directory is reserved for generated tables, figures, and reports from the ASHA survey methodology analysis.

Current outputs are generated through the root-level Quarto report files:

```text
ASHAReportFinalSLP2024.qmd
SLP2024- Preliminary Report.qmd
SLP School.qmd
```

Key output categories include:

- response-rate tables,
- postcard intervention estimates,
- bootstrap uncertainty summaries,
- mode comparison tables,
- subgroup response-rate figures,
- logistic regression summaries,
- survey-design recommendations.

Suggested future structure:

```text
outputs/
  tables/
  figures/
  reports/
```
