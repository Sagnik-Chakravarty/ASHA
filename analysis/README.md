# Analysis

This directory is reserved for cleaned and renamed analysis workflows.

The current project analysis remains in the original root-level Quarto files:

```text
ASHAReportFinalSLP2024.qmd
SLP2024- Preliminary Report.qmd
SLP School.qmd
```

These files contain analysis for:

- postcard reminder effects,
- sequential response-rate estimands,
- bootstrap inference,
- short-term follow-up effects,
- postal mail vs. web/SurveyMonkey mode comparisons,
- subgroup response-rate comparisons,
- chi-square tests,
- logistic regression models,
- stakeholder-facing recommendations.

A future cleaned version could move these workflows into a sequential analysis directory:

```text

  02_postcard_experiment.qmd
  03_mode_effects.qmd
  04_final_report.qmd
```
