# Project Overview

## ASHA Survey Methodology Analysis

This repository documents a survey methodology analysis for the American Speech-Language-Hearing Association (ASHA). The project evaluates survey design decisions related to survey mode and follow-up contact strategy, with a focus on response rates, nonresponse behavior, and subgroup variation.

## Research Motivation

Professional association surveys often require careful design choices about contact mode, follow-up reminders, and respondent burden. Small differences in design can affect response rates, subgroup participation, cost efficiency, and the quality of estimates. This project evaluates two major design questions: whether postcard reminders improve response and whether postal mail or web administration produces higher response rates.

## Research Questions

1. Do postcard reminders increase survey response rates?
2. What is the marginal effect of the first and second postcard reminder?
3. Are short-term response boosts visible after postcard mailings?
4. How do postal mail and web modes compare in response rates?
5. Do mode effects differ by age, employment status, region, or facility type?
6. Is there evidence that survey mode changes respondent composition?

## Data Sources

The project uses ASHA survey files, including respondent records and full sample-frame files from SLP surveys.

Key data contexts include:

- 2025 SLP Health Care Survey postcard experiment,
- 2024 SLP Schools Survey mixed-mode design,
- respondent file and full sample file comparisons,
- subgroup variables such as age, employment status, facility type, and region.

## Main Contributions

- Applies a survey-methodology framework to evaluate nonresponse and mode effects.
- Uses a policy estimand approach to avoid naive comparisons in postcard follow-up analysis.
- Compares response rates by mode and subgroup.
- Uses chi-square tests, logistic regression, interaction tests, and bootstrap inference.
- Produces stakeholder-facing recommendations for future survey design.

## Main Outputs

- Response-rate comparisons for postal mail vs. web administration.
- Postcard reminder effect estimates.
- Subgroup mode-effect tables and visualizations.
- Logistic regression summaries.
- Practical recommendations for survey design and follow-up strategy.
